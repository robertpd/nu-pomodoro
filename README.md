## Getting Started

Requirement:

- NodeJS 0.12+


Install dependencies:

```
npm install
```

## Running development server

Run gulp:

```
gulp watch
```

Run server:

```
npm run start-dev
```

## Testing

```
npm test
```


## Deploying to production

1. Build production assets:

    ```
    npm run build
    ```

2. Deploy to Heroku:

    ```
    heroku push origin master
    ```
